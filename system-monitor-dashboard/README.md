<h1 style="text-align: center; font-weight: bold">system-monitor-dashboard</h1>

1. 创建 Vue 工程
    ```sh
    npm create vue@latest
    ```
2. 安装依赖
    ```sh
    npm install
    ```
3. 安装 Axios ECharts Ant-Design-Vue
    ```sh
    npm install axios echarts ant-design-vue@4.x
    ```
4. 格式化代码
    ```sh
    npm run format
    ```
5. 启动项目
    ```sh
    npm run dev
    ```
6. 通过浏览器打开

    http://localhost:6061/
