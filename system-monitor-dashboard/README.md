<h1 style="text-align: center; font-weight: bold">system-monitor-dashboard</h1>

1. 安装依赖

```sh
  npm install
```

2. 安装 Axios Echarts Ant-Design-Vue

```sh
  npm install axios echarts ant-design-vue@4.x
```

3. 格式化代码

```sh
  npm run format
```

4. 启动项目

```sh
  npm run dev
```

5. 通过浏览器打开

http://localhost:5173/
