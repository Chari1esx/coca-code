<h1 style="text-align: center; font-weight: bold">mbti-test-taro</h1>

1. 安装 Taro 3.x
    ```sh
    npm install -g @tarojs/cli@3
    ```
2. 初始化 Taro 模板
    ```sh
    taro init mbti-test-taro
    ```
3. 安装 Taro UI Vue3
    ```sh
    npm install taro-ui-vue3
    ```
4. 启动项目
    ```sh
    npm run dev:weapp
    ```
5. 在微信开发者工具中进行调试
